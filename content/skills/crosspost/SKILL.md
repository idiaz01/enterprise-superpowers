# Crosspost

Distribute content across X, LinkedIn, Threads, and Bluesky with platform-specific adaptations.

## When to Use

- Posting content to multiple social platforms
- Adapting a single message for different platform conventions
- Distributing announcements, launches, or updates cross-platform

## Core Rule

Never post identical content cross-platform. Each platform gets a native adaptation.

## Platform Specifications

| Platform | Char Limit | Notes |
|----------|-----------|-------|
| X | 280 (4000 Premium) | Minimal hashtags, supports images and video |
| LinkedIn | 3000 | 3-5 hashtags, links don't count toward length |
| Threads | 500 | Separate link attachment, conversational tone |
| Bluesky | 300 | Community-oriented, uses feeds instead of hashtags |

## Workflow

1. Create source content identifying the core message
2. Identify target platforms by priority
3. Adapt content per platform conventions
4. Post to the primary platform first
5. Stagger secondary platform posts with 30-60 minute gaps

## Quality Checklist

Before posting, verify that each version:
- Reads naturally for that platform
- Respects length limits
- Matches platform conventions
- Includes appropriately sized media
